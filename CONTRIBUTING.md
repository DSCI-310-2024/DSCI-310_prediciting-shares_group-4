# Contrubuting to this project

Contributions to this project are welcome since we realize the importance of change and improvement and want to provide the best version of the project with the most useful results.

## Ways to contribute

- Make suggestions for improvement
- Report bugs
- Suggest desired features
- Help review and test the code

## Code of Conduct

Please ensure that you follow the code of conduct stated in the CODE_OF_CONDUCT.md file in the project repository root.

## Process for submitting code

- Reach out to a core group member about your object of interest - bug fix, new feature, any other suggestions.
- Create a separate branch and work on the code and submit a pull request.
- All pull requests will be reviewed through a fair and thorough process and the contributor will be credited in an appropriate way on a case by case basis.
